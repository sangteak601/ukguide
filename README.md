# 영국 생존 가이드 🇬🇧

한인들을 위한 영국 생존 가이드 — 출국 전 준비부터 정착, 생활, 안전, 문화까지.

**[https://ukguide.shop](https://ukguide.shop)**

## 내용 구성

- **출국 전 준비** — 비자, 항공권, 짐싸기
- **입국 및 초기 정착** — 입국 심사, 단기 숙소, 초기 셋업
- **생활** — 집 구하기, 교통, 통신, 계좌, 장보기, 절약
- **안전** — 의료(NHS), 치안, 날씨
- **문화** — 영국 문화와 한국과의 차이점

## 기여하기

틀린 내용을 발견하거나 추가하고 싶은 내용이 있다면 [내용 추가/변경 요청하기](https://salt-lace-abb.notion.site/341e6f37e0af80e88d12fc06d0ef4ea7?pvs=105)를 통해 알려주세요.

## 로컬 개발

[Jekyll](https://jekyllrb.com)과 [Bundler](https://bundler.io)가 설치되어 있어야 합니다.

```bash
bundle install
bundle exec jekyll serve
```

브라우저에서 `http://localhost:4000`으로 접속하면 사이트를 미리 볼 수 있습니다.
